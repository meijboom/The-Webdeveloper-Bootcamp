How to play:

The game is played by guessing the correct colored square on the page. this is done by deducting the RGB colorcode. 0 means there is no color present, 255 is the maximum amount of that color present.

  • The first number RGB( 255, X , X ) is Red
  • The second number RGB( X , 255, X ) is Green
  • The third number RGB( X , X , 255) is Blue
